# 🌸 Cuida+

O **Cuida+** é um aplicativo mobile e web desenvolvido com React Native e Expo go web, feito especialmente para acompanhar e apoiar gestantes durante a jornada da gravidez. Com um design fofinho, acolhedor e limpo, o app foi pensado para transmitir cuidado, segurança e calmaria em cada clique.

## ✨ Funcionalidades

*   **Tela Inicial Acolhedora:** Uma mensagem de boas-vindas fofinha para iniciar a experiência com afeto.
*   **Fluxo de Login:** Proteção simples e integrada para a usuária acessar seus dados.
*   **Acompanhamento de Sintomas:** Registro inteligente de sintomas com mensagens de aconchego para o dia a dia.
*   **Calendário de Compromissos:** Organização de consultas e vacinas, permitindo marcar as tarefas como concluídas diretamente no card.
*   **Comunidade privada:**  Espaço exclusivo e anônimo onde gestantes podem trocar experiências, compartilhar dúvidas e oferecer apoio mútuo.

---

## 🎨 Identidade Visual

O design do aplicativo utiliza uma paleta de cores pastel e formas arredondadas para reforçar o sentimento de cuidado e conforto:
*   **Rosa Pastel (`#FFF0F2`):** Utilizado nos fundos para evitar o cansaço visual.
*   **Vermelho Suave (`#cc4b4b`):** Cor principal para destaques, botões de ação e títulos.
*   **Rosa Antigo (`#aa7078`):** Utilizado em textos secundários e legendas.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando:
*   [ React Native ]
*   [ Expo go web ]
*   [ JavaScript ]

> **Nota de Desenvolvimento:** O fluxo de navegação foi otimizado utilizando Estados Nativos (`useState`) do React, garantindo compatibilidade de 100% e transições de tela sem travamentos tanto no ambiente mobile quanto no **Expo Go Web**.
